
# Employee Attendance Tracker

The Employee Attendance Tracker is an interactive Microsoft Excel dashboard designed to monitor employee attendance, working hours, leave records, and attendance trends.



## Features 
-Department wise Attendance
-Month Wise Attendance
-Department Wise Absent Days
-Leave By Department
-Attendance By Employees
-KPI
-Slicers


## Tools Used

-Ms Excel
-Pivot Table
-Pivot Chart
-Slicers 
-Conditional Formatting 
-kPI
-Filters
